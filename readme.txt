This plugin let's you easily insert Altery snippet code in the header of your WordPress blog.

== Description ==

[Altery](https://www.alterly.com/ "Altery") is a service that let you easily edit all parts of your website.

The Alterly WordPress plugin is a simple plugin that lets you add your personal Alterly snippet code to your WordPress site without having to get your hands dirty in code.

== Installation ==

#### Get a free Alterly account and ID

1. Go to the [Altery](https://www.alterly.com/ "Altery") website and sign up for a free account
2. Go to your account [profile page](https://www.alterly.com/en/app/profile "profile page") and copy your Alterly account ID

#### Upload, activate and configure the plugin

1. Upload the `alterly` directory to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to the 'Settings/Alterly' menu in WordPress
4. Paste your account ID the box, click Save and you're site will be ready to use with Alterly
